# Bidra til prosjektet

Velkommen! Dette prosjektet bruker Next.js med TypeScript, ESLint, Prettier og en struktur som legger til rette for høy kodekvalitet og Copilot-drevet utvikling.

## Mappestruktur

- `app/` – Sider og rotkomponenter (Next.js App Router)
- `components/` – Gjenbrukbare React-komponenter
- `lib/` – Felles logikk, API-klienter, hooks, utils
- `styles/` – Globale og modulspesifikke stilark
- `public/img/` – Bilder og statiske filer

## Kodekvalitet og konvensjoner

- **Språk:** All kode, UI og dokumentasjon skal være på norsk.
- **TypeScript:** Bruk strenge typer og unngå `any`.
- **ESLint:** Kjør `pnpm lint` før du lager en pull request.
- **Prettier:** Kjør `pnpm format` for å sikre lik formattering.
- **Copilot:** Bruk Copilot aktivt, men gjennomgå alltid forslagene kritisk.
- **Sanity:** Kodebasen skal være klar for integrasjon med Sanity (se egen guide når dette er aktuelt).

## Pull requests

- Beskriv endringer tydelig på norsk.
- Kjør alle tester og lintere før du sender inn.
